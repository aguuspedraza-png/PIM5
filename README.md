# PIM5 - Proyecto Integrador M5

Proyecto integrador de ciencia de datos para una empresa financiera ficticia. 
Consiste en el desarrollo de un modelo predictivo de scoring crediticio, simulando 
un pipeline de MLOps completo: carga de datos, analisis exploratorio, feature 
engineering, entrenamiento y evaluacion de modelos, despliegue y monitoreo.

## Estructura del repositorio

- `mlops_pipeline/src/`: notebooks y scripts del pipeline
- `Base_de_datos.csv`: dataset de solicitudes de credito
- `requirements.txt`: dependencias del proyecto

## Ramas

- `developer`: desarrollo activo
- `certification`: validacion previa a produccion
- `master`: version estable
